Eddig projektjeink alatt használt technológiák

ASP.NET MVC

Belső használatra szánt adminisztratív, reporing, ERP témájú weboldalak létrehozása .NET keretrendszerben és modern webes technológiák felhasználásával.

HTML 5

Reszponzív weboldalak fejlesztése a piacon elérhető legmodernebb technológiák segítségével - GulpJs, WebPack, TypeScript, AngularJS, NodeJS.

.NET C#

Egyedi új szoftverek, meglévő rendszerek továbbfejlesztése .NET keretrendszerben, Windows Presentation Foundation technológia használatával.

Android

Mobil alkalmazások fejlesztése napjaink legelterjedtebb mobil operációs rendszerére a legfrissebb Android komponensek és Google API-k felhasználásával.

UWP

A Microsoft legfrissebb technológiája. Lehetővé teszi, hogy a megírt alkalmazásainkat PC-n, telefonon, Xbox-on és IoT eszközön is futtathassuk.

Unity

Látványos 2D és 3D cross-platform (Android, iOS, Windows Store, Xbox One, stb.) játékok fejlesztése a világ legelterjedtebb játékfejlesztő motorjával.
